# 电脑端项目文件索引

## 电脑端组成

```text
schedule/
├─ schedule_backend/
├─ schedule_frontend/
├─ build-release.ps1
├─ build-release.bat
├─ start-dev.ps1
├─ stop-dev.ps1
├─ release/
├─ 电脑端任务记忆文档.md
├─ 电脑端项目文件索引.md
└─ 电脑端项目文件索引.json
```

## 后端

路径：`schedule_backend/`

关键文件：

- `run.py`：后端启动入口，支持 PyInstaller sidecar 无控制台运行。
- `app/main.py`：FastAPI 应用入口。
- `app/core/config.py`：配置入口，支持 `SCHEDULE_HOST`、`SCHEDULE_PORT`、`SCHEDULE_DATA_DIR`。
- `app/core/database.py`：SQLite 初始化。
- `app/api/`：业务 API。
- `app/services/`：业务逻辑。
- `app/services/sync_service.py`：本地云同步编排和远端 apply。
- `app/services/sync_mapper.py`：sync envelope 映射。
- `tests/`：后端测试。

## 前端

路径：`schedule_frontend/`

关键文件：

- `package.json`：前端脚本，包含 `build:desktop` 和 `tauri:build`。
- `.env.desktop`：安装版 API 地址，固定为 `http://127.0.0.1:18765`。
- `src/main.ts`：前端入口。
- `src/router/index.ts`：路由。
- `src/pages/TodayPage.vue`：今日任务。
- `src/pages/LongTermTasksPage.vue`：长期任务。
- `src/pages/SchedulePage.vue`：日程。
- `src/pages/StudyRecordsPage.vue`：学习记录。
- `src/pages/SettingsPage.vue`：设置与云同步。
- `src/pages/AiWorkspacePage.vue`：AI 工作区。
- `src/stores/sync.ts`：云同步前端状态。

## Tauri

路径：`schedule_frontend/src-tauri/`

关键文件：

- `tauri.conf.json`：Tauri 配置，包含 NSIS、sidecar、窗口配置。
- `Cargo.toml`：Rust 依赖，包含 `tauri-plugin-shell`。
- `src/main.rs`：Tauri 入口，自动启动/关闭内置后端。
- `binaries/.gitignore`：忽略打包生成的 sidecar exe。
- `binaries/schedule-backend-x86_64-pc-windows-msvc.exe`：构建时生成，不提交源码仓库。

## 发布脚本

- `build-release.ps1`：一键构建发布包。
- `build-release.bat`：双击入口。

构建产物：

- `release/Schedule_0.4.0_x64_setup.exe`
- `schedule_frontend/src-tauri/target/release/bundle/nsis/日程安排_0.4.0_x64-setup.exe`

## 开发脚本

- `start-dev.ps1`：一键启动开发环境。
- `start-dev.bat`：双击开发启动入口。
- `stop-dev.ps1`：停止开发进程。
- `stop-dev.bat`：双击停止入口。
- `scripts/start-backend.ps1`
- `scripts/start-frontend.ps1`
