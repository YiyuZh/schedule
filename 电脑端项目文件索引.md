# 电脑端项目文件索引

## 电脑端组成

```text
schedule/
├─ schedule_backend/
├─ schedule_frontend/
├─ start-dev.ps1
├─ stop-dev.ps1
├─ 电脑端任务记忆文档.md
├─ 电脑端项目文件索引.md
└─ 电脑端项目文件索引.json
```

## 后端

路径：`schedule_backend/`

关键文件：

- `run.py`：后端启动入口。
- `app/main.py`：FastAPI 应用入口。
- `app/core/database.py`：SQLite 初始化。
- `app/api/`：业务 API。
- `app/models/`：SQLAlchemy 模型。
- `app/schemas/`：Pydantic schema。
- `app/services/`：业务逻辑。
- `app/api/sync.py`：本地同步 API。
- `app/services/sync_service.py`：本地同步编排和远端 apply。
- `app/services/sync_mapper.py`：sync envelope 映射。
- `tests/`：后端测试。

后端细分索引：

- `schedule_backend/任务记忆文档.md`
- `schedule_backend/项目文件索引.md`
- `schedule_backend/项目文件索引.json`

## 前端

路径：`schedule_frontend/`

关键文件：

- `src/main.ts`：入口。
- `src/App.vue`：根组件。
- `src/router/index.ts`：路由。
- `src/layouts/AppLayout.vue`：桌面布局。
- `src/pages/TodayPage.vue`：今日任务。
- `src/pages/LongTermTasksPage.vue`：长期任务。
- `src/pages/SchedulePage.vue`：日程。
- `src/pages/StudyRecordsPage.vue`：学习记录。
- `src/pages/ImportPage.vue`：导入中心。
- `src/pages/SettingsPage.vue`：设置和云同步。
- `src/pages/AiWorkspacePage.vue`：AI 工作区。
- `src/stores/sync.ts`：云同步前端状态。
- `src/api/sync.ts`：同步 API 封装。

## Tauri

路径：`schedule_frontend/src-tauri/`

关键文件：

- `tauri.conf.json`：Tauri 配置。
- `src/main.rs`：Tauri Rust 入口。
- `.cargo/config.toml`：Cargo 配置。

当前打包路线：

- Windows 安装包使用 NSIS。
- 不恢复 WiX MSI。

## 启动脚本

- `start-dev.ps1`：一键启动后端和前端。
- `start-dev.bat`：双击入口。
- `stop-dev.ps1`：停止开发进程。
- `stop-dev.bat`：双击停止入口。
- `scripts/start-backend.ps1`
- `scripts/start-frontend.ps1`
